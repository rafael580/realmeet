

<h1>Projeto real meet</h1>

<h2>Projeto real meet vem para solucionar o problema 
de gerencimento de salas de reunião</h2>

<h3>Entendendo requisitos do sistema</h3>
<p>software, gerência de horários de salas de reunião </p>
<p>Cadastro de salas deve ser feito dentro do real meet</p>
<p>Cada sala terá: nome, numero de lugares disponíveis</p>
<p>Real meet pode cadastrar, deletar, excluir e listar salas de reunião, de acordo com o 
critério da pesquisa</p>
<p>Alocação da sala é feito com horário e data de início, os dados para cadastrar são
nome e e-mail, também é possivel fazer uma descrição da reunião</p>
<p>Cada agendamento pode ser feito com no máximo 4 horas de antecedência</p>
<p>Real meet gera relatórios e envio de e-mails</p>
